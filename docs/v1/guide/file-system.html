文件系统

CatLib提供的文件系统抽象允许使用一个API来对多个本地(云存储)的文件存储系统进行操作。

CatLib文件系统已经集成了处理本地文件系统的简单驱动。

上下文环境

如果没有特殊的说明，本文中的示例项目都是基于下述上下文环境，默认使用本地文件驱动：

您可以通过IFileSystemManager的Disk()方法来获取文件系统的磁盘实例。

默认的磁盘

CatLib默认的磁盘会被定位到Env.AssetPath。详情请参考CatLib核心 / 环境

文件系统API

Read

通过Read()方法可以读取文件的字节流。

Exists

通过Exists()可以判断文件是否存在。

Delete

通过Delete()可以删除文件。

Write

通过Write()可以写入一个文件，如果文件已经存在那么覆盖数据。

Move

移动文件或文件夹到指定路径中。

Move()可以被用于重命名。

Copy

复制文件或文件夹到指定路径。

MakeDir

新建一个文件夹。

GetAttributes

获取文件或文件夹的属性。

GetSize

获取文件或文件夹的大小（如果是文件夹则是总大小）。

GetHandler

获取文件或文件夹的句柄（可以通过句柄直接操作这个文件或文件夹）

GetList

通过GetList()可以获取指定路径下的所有文件夹和文件的句柄(注意不会迭代子文件夹)。

自定义文件系统

CatLib文件系统允许您自定义您的文件系统驱动，您只需要使用Extend()就可以自定义拓展。

扩展后您可以通过Disk()来获取扩展实例。
